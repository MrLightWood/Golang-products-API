Welcome to products API

Here are available API endpoints

Category:

  • GET /api/categories | Get all categories
  • GET /api/categories/:id | Get category of id id
  • POST /api/categories | create a category. Send value "name: string" as JSON in body
  • PUT /api/categories/:id | update a category of id id. Send value "name: string" as JSON in body
  • DELETE /api/categories/:id | delete a category of id id

Product:

  • GET /api/products | Get all products
  • GET /api/products/:id | Get product of id id
  • POST /api/products | create a category. Send value "name: string", "description: string", "category: int", "price: int" as JSON in body
  • PUT /api/products/:id | update a category of id id Send value "name: string", "description: string", "category: int", "price: int" as JSON in body
  • DELETE /api/products/:id | delete a category of id id
